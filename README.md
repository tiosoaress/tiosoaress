## Sobre mim

Sou **Desenvolvedor Full-Stack e estudante de Ciência da Computação**, com foco em projetar, desenvolver e evoluir softwares que resolvem problemas operacionais reais.

Meu trabalho é concentrado em **engenharia de software, plataformas internas, automação de fluxos de trabalho, sistemas orientados a dados e desenvolvimento de produtos**. Minha abordagem de engenharia vai além de funcionalidades isoladas: busco compreender o processo de negócio, projetar a arquitetura, implementar a solução e melhorar continuamente o produto final.

Desenvolvo aplicações full-stack com forte atenção à **arquitetura de sistemas, engenharia de backend, bancos de dados relacionais, experiência de frontend, permissões, integrações, automação, desempenho e manutenibilidade**.

Também exploro ativamente **Inteligência Artificial, Grandes Modelos de Linguagem, infraestrutura local de IA, desenvolvimento assistido por IA e experiências de produto inteligentes**, com foco em integrar a IA a softwares práticos, em vez de tratá-la como um experimento isolado.

Minha visão de engenharia combina:

* **Engenharia de Software** — arquiteturas sustentáveis, bases de código organizadas e sistemas confiáveis.
* **Desenvolvimento Full-Stack** — entrega completa de produtos envolvendo frontend, backend, bancos de dados e infraestrutura.
* **Engenharia de Produto** — transformação de fluxos de negócio em produtos digitais úteis e intuitivos.
* **Engenharia de IA** — exploração de funcionalidades baseadas em LLMs, fluxos inteligentes, modelos locais e ferramentas para desenvolvedores.
* **Automação** — redução de processos repetitivos e integração de sistemas por meio de fluxos eficientes.

### Aberto a oportunidades em

`Engenharia de Software` · `Desenvolvimento Full-Stack` · `Engenharia de Produto` · `Engenharia de IA` · `Colaboração Open Source`

## Projetos em destaque

<details>
<summary><b>Portal Dev — Plataforma centralizada de operações de engenharia</b></summary>

<br>

Uma plataforma interna centralizada de engenharia, desenvolvida para consolidar operações de desenvolvimento, visibilidade dos sistemas, gerenciamento de acesso de usuários, cargos, permissões e fluxos técnicos.

| Métrica         | Detalhes                                                               |
| :-------------- | :--------------------------------------------------------------------- |
| **Stack**       | JavaScript · Python · SQL Server · APIs REST                           |
| **Escala**      | Ambiente interno de engenharia com múltiplos sistemas                  |
| **Desempenho**  | Acesso centralizado a dados operacionais e informações das plataformas |
| **Segurança**   | Controle de acesso baseado em cargos e gerenciamento de permissões     |
| **Impacto**     | Reduz processos fragmentados e centraliza a governança técnica         |
| **Repositório** | Privado / Interno                                                      |

O projeto segue uma abordagem de engenharia de plataformas: em vez de manter permissões, ferramentas operacionais e processos de engenharia distribuídos entre sistemas desconectados, o objetivo é oferecer um ambiente interno estruturado no qual desenvolvedores possam gerenciar e monitorar recursos essenciais em um único local.

</details>

<br>

<details>
<summary><b>RPTools — Plataforma interna de produtividade para desenvolvedores</b></summary>

<br>

Um sistema de produtividade de engenharia criado para centralizar ferramentas internas, utilitários operacionais, jobs automatizados, ações em sistemas e fluxos recorrentes de desenvolvimento.

| Métrica         | Detalhes                                                            |
| :-------------- | :------------------------------------------------------------------ |
| **Stack**       | Web Full-Stack · Python · JavaScript · SQL                          |
| **Escala**      | Ecossistema de ferramentas internas                                 |
| **Desempenho**  | Acesso mais rápido a operações técnicas repetitivas                 |
| **Segurança**   | Acesso controlado a ações e recursos internos                       |
| **Impacto**     | Reduz o trabalho manual e padroniza as operações de desenvolvimento |
| **Repositório** | Privado / Interno                                                   |

O RPTools é desenvolvido com base em um princípio simples: trabalhos técnicos repetitivos devem ser transformados em fluxos estruturados de produto.

A plataforma organiza utilitários de engenharia em um ambiente estruturado, com ações claras, descrições compreensíveis, permissões controladas e automações reutilizáveis.

</details>

<br>

<details>
<summary><b>Lume Atelier — Plataforma empresarial de iluminação decorativa</b></summary>

<br>

Uma plataforma digital completa para uma empresa de iluminação decorativa, combinando uma experiência comercial pública com um sistema interno de gestão operacional.

| Métrica         | Detalhes                                                                          |
| :-------------- | :-------------------------------------------------------------------------------- |
| **Stack**       | Web Full-Stack · Interface moderna · Gestão de fluxos de trabalho                 |
| **Escala**      | Site público e plataforma empresarial interna                                     |
| **Desempenho**  | Fluxo comercial e operacional unificado                                           |
| **Segurança**   | Separação entre a experiência pública e o ambiente interno autenticado            |
| **Impacto**     | Centraliza orçamentos, propostas, contratos, agendamentos e operações de projetos |
| **Repositório** | Projeto privado                                                                   |

A plataforma conecta toda a jornada do negócio: apresentação da empresa, portfólio, contato com clientes, solicitações de orçamento, criação de propostas, exportação em PDF, gestão de contratos, fluxo de projetos e agendamentos.

O produto foi desenvolvido para eliminar a fragmentação de informações entre planilhas, mensagens, documentos e ferramentas desconectadas.

</details>

<br>

<details>
<summary><b>Timesheet Intelligence — Fluxo de dados operacionais e relatórios</b></summary>

<br>

Um fluxo orientado a dados para analisar registros de horas, alocação em projetos, atividade de funcionários, lançamentos pendentes e relatórios gerenciais.

| Métrica         | Detalhes                                                           |
| :-------------- | :----------------------------------------------------------------- |
| **Stack**       | SQL Server · Python · Automação com Excel · Processamento de dados |
| **Escala**      | Conjuntos de dados de funcionários, projetos e relatórios mensais  |
| **Desempenho**  | Consolidação automatizada de registros operacionais                |
| **Segurança**   | Processamento interno de dados empresariais                        |
| **Impacto**     | Reduz análises manuais e melhora a visibilidade dos relatórios     |
| **Repositório** | Privado / Interno                                                  |

O fluxo transforma registros brutos de horas em resultados gerenciais estruturados, incluindo resumos por funcionário, detalhamento por projeto, identificação de registros ausentes, análises mensais, relatórios expansíveis e exportações de planilhas com formatação profissional.

</details>
